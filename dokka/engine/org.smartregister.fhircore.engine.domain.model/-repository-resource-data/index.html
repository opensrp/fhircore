val queryResult: RepositoryResourceData.QueryResult

The response returned from the database query. The response can either be a QueryResult.Search or QueryResult.Count. QueryResult.Search includes a Resource and optionally a list of related RepositoryResourceData, typically this result is produced when a SELECT query returns FHIR Resource's. QueryResult.Count represents the count for the configured resources
